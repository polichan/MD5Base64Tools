# MD5Base64Tools
Base64编码后的MD5字符串处理工具
### 使用方法:
#### 例1：
解 Base64 编码后的原 MD5
```java
public static void main(String[] args) throws Exception
{
  System.out.println(decodeMd5EncryptBase64("在这输入字符串"));
}
```
